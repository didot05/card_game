For the game to run, you must have JavaFX installed.
If you do not have JavaFX installed, then you may find https://openjfx.io/openjfx-docs/ useful for installation.

The code conforms to the Intellij Project Structure. 

Inside every prototype folder, there is a folder called "Software Engineering". You should use this folder as the root project folder
in your IDE.

Instructions To Run The Game:
Run the main method in "Main.java".
